## Reserba nga `id` para sa umaabot

Kini nga mga `id` gireseerba para sa umaabot. Tingali, madugangan ang maong listahan.

| id       | Deskripsiyon                           |
|:--------:|----------------------------------------|
| `phpt`   | Pakighisgot sa Higugmaa ang mga Tawo   |
| `pg`     | Pagpamalaybalay                        |
| `ip`     | Impormal nga Pagsangyaw                |
| `psd`    | Pasundayag                             |
